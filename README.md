# Rpal Interpreter

## Running the Programme

1. Run
```bash
java -jar rpal.jar filename
```

or

1. cd into `bin/` directory.

2. Run 
```bash
java Rpal filename
eg:  java Rpal "/home/user/Documents/code.ast"
```

Source code is in `src/` directory.
